Description

A self-contained panel for the Heads-Up Display (HUD) that shows at-a-glance information for a single player, including their name, token, and current cash balance. It supports states for active turn and bankruptcy.

Design System Alignment

This component is built as a 'Card' as per the design system, using typography, color, and spacing tokens for consistent styling. It is composed of atomic 'Icon' and 'Text Label' components.

Desktop Enhancements

Accessibility Features

Keyboard Support

N/A. This is a non-interactive data display component.

Screen Reader Support

Semantic structure allows screen readers to announce player name and cash amount. A parent element should provide context (e.g., 'Player 1 Information').

Focus Management

N/A. Non-interactive.

Aria Attributes

  • aria-live='polite' can be added to the cash element via JavaScript to announce changes.
  • aria-label on the root element can provide a comprehensive summary, e.g., 'Player 1: BoardwalkKing, Cash: $1,500'.

Color Contrast

Text colors meet WCAG AA contrast ratios against the white background. The active state uses a border and shadow, not just color, to indicate the current turn.
