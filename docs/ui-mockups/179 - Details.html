Responsive Specifications

Desktop Layout

  • Centered modal-style layout with a max-width.
  • A responsive grid for save slots that adjusts the number of columns based on screen width.
  • Footer actions are anchored to the bottom right.

Accessibility Features

Keyboard Navigation

Complete tab order through all save slots and action buttons. Arrow keys can be used to navigate the grid of save slots.

Screen Reader Support

Semantic markup with H1 for the title. Each slot will have an aria-label describing its state and content (e.g., 'Save Slot 1, Occupied, Player BoardwalkKing, Turn 45').

Focus Management

A clear, high-contrast focus ring is applied to the active element. When a slot is selected, focus moves to the primary action button.

Aria Attributes

  • aria-label
  • aria-disabled
  • role='dialog'
  • aria-modal='true'

Color Contrast

All text and interactive elements meet WCAG 2.1 AA contrast ratios. Invalid states (corrupted, incompatible) use icons and text in addition to color.
