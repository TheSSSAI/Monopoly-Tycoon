Responsive Specifications

Desktop Layout

  • A large, centered modal dialog overlays the game screen.
  • The Game Summary variant uses a flexbox or grid layout to accommodate 2, 3, or 4 player stat cards.
  • Layout is constrained by a max-width to ensure readability on ultrawide monitors.

Tablet Adaptations

  • N/A - Desktop only application.

Mobile Adaptations

  • N/A - Desktop only application.

Accessibility Features

Keyboard Navigation

Logical tab order from the main message to the action buttons.

Screen Reader Support

Semantic headings (h1, h2) for screen titles and player names. Role attributes on modal dialogs.

Focus Management

Focus is trapped within the modal dialog until an action is taken. A clear focus indicator is present on buttons.

Aria Attributes

  • role='dialog'
  • aria-labelledby='dialog-title'
  • aria-modal='true'

Color Contrast

All text combinations (e.g., white text on dark blue background) meet WCAG 2.1 AA standards. Winner/loser status is indicated by icons and text, not just color.
