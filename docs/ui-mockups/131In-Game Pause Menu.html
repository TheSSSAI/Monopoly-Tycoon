Component: Player HUD Panel

This molecular component displays the real-time status of a single player. It is designed to be placed within the main Heads-Up Display (HUD) organism, typically anchored to a corner of the screen.

Component Variations

Human Player

Class: .hud-panel--human

Provides a subtle visual distinction for the human player's panel, typically by changing the background color of the token icon. This helps the user quickly identify their own status panel.

AI Player

Class: .hud-panel--ai or no specific class

The default appearance for an AI opponent. It has a neutral token background.

Active Turn

Class: .hud-panel--active

This state is crucial for gameplay. It adds a prominent glowing border and slightly scales up the panel to clearly indicate whose turn it is. This style overrides the default state.

Bankrupt

Class: .hud-panel--bankrupt

Indicates a player has been eliminated from the game. The panel is desaturated, its opacity is reduced, and the cash value is styled to show the player is out of money. The "BANKRUPT" label appears above the cash value for clarity.
