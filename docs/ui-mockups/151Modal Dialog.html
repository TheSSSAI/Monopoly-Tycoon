Modal Dialog Component

A modal dialog requires a user to interact with it before they can return to the main application view. It's used for critical, turn-halting actions and information.

Static Example (Visible by Default)

Purchase Property?

Would you like to purchase Boardwalk for $400? This property is part of the Dark Blue set.

Interactive Variants

Decision

Presents the user with multiple choices for a critical action.

Confirmation

Asks the user to confirm a destructive or irreversible action.

Error

Informs the user about an error that prevents an action.

Informational

Provides important information that must be acknowledged.

Income Tax

You have landed on Income Tax. Please choose how you would like to pay.

Reset All Statistics?

Are you sure you want to reset all your historical data? This action is permanent and cannot be undone.

Insufficient Funds

You do not have enough cash to build a house on this property. You can raise funds by mortgaging properties or selling buildings.

Building Shortage

The Bank has run out of houses. The remaining houses will now be auctioned off to eligible players.
