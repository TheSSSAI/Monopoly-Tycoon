Save/Load Slot

This component is used in the Save and Load Game menus to display the status and metadata of a persistent game file.

Occupied Slot

Empty Slot

Unusable Slots (Disabled)

Component Variations

Variant Name: Empty

Represents an available slot where a new game can be saved. Has a distinct visual style (dashed border) to differentiate it.

Variant Name: Occupied

The standard state for a valid, loadable save file. Displays all relevant metadata for the player to identify the save.

Variant Name: Corrupted

Indicates that the save file has failed an integrity check (checksum mismatch). The component is disabled to prevent load attempts.

Variant Name: Incompatible

Indicates the save file was created with an older, unsupported version of the game. The component is disabled.
