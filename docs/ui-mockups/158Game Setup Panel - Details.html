Component Name

Description

A comprehensive form panel that allows a player to configure a new game session. It includes player profile creation, token selection, and configuration of 1-3 AI opponents with individual difficulty settings.

Design System Alignment

This organism composes several atomic and molecular components, including Text Input, Segmented Control, and Buttons. It uses the design system's color palette, typography, spacing, and card styles to create a cohesive layout.

Responsive Specifications

Mobile Adaptations

  • On screens narrower than 768px, the AI opponent configuration rows stack vertically for better readability.
  • The final action button becomes full-width.

Tablet Adaptations

  • The component maintains its desktop layout.

Desktop Enhancements

  • Utilizes a centered, max-width card layout suitable for various desktop aspect ratios.
  • The token selection uses a responsive grid that adjusts the number of columns based on available space.

Accessibility Features

Keyboard Support

All interactive elements, including text inputs, segmented controls, radio buttons for tokens, and the final action button, are keyboard navigable and activatable.
